## 介绍

河南理工大学企业微信自动健康打卡

特性：
1. 自动企业微信健康打卡；
2. 微信通知打卡结果；
3. 前端自动部署每个人的信息。

## 实现

前端接收**学号** 、 **密码** 、**微信通知密钥** 等个人信息。

> 我们承诺，将**不会乱使用您的个人信息**，请放心提交！

后端部分使用模拟登录并进行打卡，同时依托*Server酱* 作为载体对打卡成功与否进行客户通知
